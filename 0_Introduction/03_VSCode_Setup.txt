Open VSCode

Create New File
  File Type: Jupyter Notebook .ipynb

Open Terminal:
  Change directory into your project folder
  $ cd
  $ cd Desktop
  $ cd project_folder

  OR create new project folder and change directory into your new project folder
  $ mkdir your_project_name
  $ cd your_project_name

Install the virtual environment
$ virtualenv -p python3 venv

Activate your virtual Environment (everytime you begin development)
  $ . venv/bin/activate
  WINDOWS:  
  $ .\venv\Scripts\Activate

Install numpy package
  $ pip install numpy

Install pandas package
  $ pip install pandas

Note: You will probably be prompted to select a kernel when you run your first code block, install if needed

Deactivate your virtual Environment
  $ deactivate

You will be prompted to select a kernel when you run your first code block, install if needed
